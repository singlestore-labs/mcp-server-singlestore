# [0.4.16] - 2026-03-26

## Added

- `stage_list_files` tool to list files and folders in a Stage deployment's file system
- `stage_get_file` tool to get a file from Stage by path, returning metadata, a download URL, or text content
- `stage_create_folder` tool to create a folder in Stage
- `stage_upload_file` tool to upload a file to Stage with text content
- `stage_move` tool to move or rename a file or folder in Stage
- `stage_delete` tool to delete a file or folder from Stage
