# [0.4.19] - 2026-06-09

## Added

- `list_cloud_functions` tool to list cloud functions in the current organization, with optional pagination
- `get_cloud_function` tool to get detailed information about a specific cloud function
- `create_cloud_function` tool to deploy a notebook from Stage as a serverless HTTP endpoint
- `update_cloud_function` tool to update a cloud function's configuration
- `delete_cloud_function` tool to delete a cloud function by ID
- `get_cloud_function_token` tool to retrieve a JWT token for invoking a cloud function endpoint
